# Linux Notes

A collection of notes I write while exploring Linux.

This repository isn't meant to be a complete Linux guide or a tutorial. Instead, it's a place where I document topics that I find interesting, concepts I wanted to understand better, and things I learned while working on servers, debugging issues, building infrastructure, or simply reading about how Linux works.

Most notes start with a question I had or a problem I encountered. The goal is to keep everything practical, concise, and easy to revisit later.

## What you'll find here

* Linux internals
* Process management
* Filesystems
* Networking
* Containers
* System administration
* Performance
* Security
* Anything else that makes me stop and think, *"How does this actually work?"*

## Why this repository exists

I often forget small details after a few months. Writing things down helps me understand them better and gives me a place to return to when I need a quick refresher.

If these notes happen to help someone else along the way, that's a nice bonus.
